## 安装使用方法

安装nodejs后，然后使用npm安装docdify

```
npm i docsify-cli -g
```

git clone后进入目录，

```
docsify serve
```

然后显示本地地址

PS C:\Users\Administrator\Documents\rk3399pro> docsify serve

Serving C:\Users\Administrator\Documents\rk3399pro now.

Listening at http://localhost:3000
